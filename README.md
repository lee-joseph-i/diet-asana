
###  The Dietsana project is a built-from-scratch clone of Asana. Asana is a web and mobile application designed to help teams organize, track, and manage their work.

### Link to live site: 

  https://dietsana.herokuapp.com/


### Technologies used:

  Backend: 
  Ruby on Rails
  
  Frontend: 
  React (Javascript)
  
  Notable Libraries: 
  JQuery
